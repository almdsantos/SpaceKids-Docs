## 📖 Sobre o SpaceKids

O **SpaceKids** é uma plataforma de entretenimento e educação infantil desenvolvida pela **JAM Technologies**. Com uma identidade visual futurista inspirada na exploração espacial, o aplicativo organiza de forma simples e segura conteúdos como jogos, vídeos, séries e experiências educativas para crianças.

O projeto foi pensado para ser intuitivo, acessível e completamente seguro — uma nave espacial onde cada criança é o astronauta da própria jornada de aprendizado e diversão.

> *"Uma galáxia de conteúdos, a um toque de distância."*

---

## 🎯 Objetivos

- Centralizar e organizar conteúdos infantis de qualidade em um único lugar
- Oferecer uma experiência segura, sem coleta de dados pessoais de crianças
- Proporcionar uma interface intuitiva tanto para crianças quanto para pais e responsáveis
- Servir como hub institucional da JAM Technologies e do ecossistema SpaceKids
- Evoluir para um portal completo com suporte, FAQ e recursos educativos

## 💻 Como Executar Localmente

Este repositório contém páginas HTML estáticas. Não há dependências de build obrigatórias.

### Pré-requisitos

- Git instalado
- Qualquer servidor HTTP local (opcional, mas recomendado)

### Passos

**1. Clone o repositório**

```bash
git clone https://github.com/jam-technologies/spacekids-docs.git
cd spacekids-docs
```

**2. Abra diretamente no navegador**

```bash
# Linux / macOS
open privacidade.html

# Ou simplesmente arraste o arquivo para o navegador
```

**3. Ou use um servidor local (recomendado)**

```bash
# Com Python
python3 -m http.server 8080

# Com Node.js (http-server)
npx http-server .
```

Acesse `http://localhost:8080` no navegador.
